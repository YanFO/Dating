Features Specification
本系統包含三大核心功能模組：

功能一：情境感知實體破冰輔助 (Context-Aware Icebreaker)
說明：針對實體生活中的搭訕需求。

輸入：使用者透過文字描述或上傳當下場景與對象特徵的照片（例如：咖啡廳、對方的穿搭）。

處理：利用視覺辨識與大型語言模型分析環境線索。

輸出：生成符合當下物理環境的自然開場白，並提供非語文行為建議（如：保持社交距離、微蹲等）。

功能二：深度語意與情緒解析之訊息回覆引擎 (Semantic & Emotion Reply Engine)
說明：解決數位聊天（如 LINE、Tinder）中的回覆焦慮。

輸入：聊天紀錄的螢幕截圖或純文字對話脈絡。

處理：進行細粒度情感分析與潛台詞解碼，判斷對方是生氣、敷衍還是正在進行社交測試。

輸出：

多種語氣風格（幽默、真誠、調情等）的回覆選項。

「教練解析面板」：向使用者解釋為何建議這樣回覆，以及當下應採取的戰略（如：冷凍處理、降低需求感）。

功能三：即時對話分析與語音導航教練 (Real-time Voice Coaching)
說明：在語音通話或約會當下提供即時指導。

輸入：透過麥克風擷取雙方的即時語音串流。

處理：進行超低延遲的語音轉文字（STT）與聲學情緒分析，偵測對話是否出現尷尬空白或不耐煩語氣。

輸出：在畫面上即時閃爍提示，推播適合當下的互動式破冰問題或話題轉移建議。
